# fmdscp-web
Frontend to fmdscp

Simple DICOM SCP

## Requirements
- MySQL server
- This has only been tested on Windows.
- Currently using node 6.4.0 but higher should work.

## Running
```
npm install
npm run start
```

- browse to http://localhost:3000/
- browse to http://localhost:3000/setupsystem to create the database
